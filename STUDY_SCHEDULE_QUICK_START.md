# 学习日程功能快速开始指南

## 功能已完成 ✅

学习日程管理功能已完整实现并集成到系统中！

## 快速测试步骤

### 1. 启动后端服务器

```bash
cd exam/backend
python run.py
```

后端服务器将在 `http://localhost:5000` 启动

### 2. 启动前端开发服务器

```bash
cd exam/frontend
npm run dev
```

前端服务器将在 `http://localhost:5173` 启动

### 3. 访问学习日程功能

有两种方式访问：

#### 方式1：从 Dashboard 查看今日日程
1. 登录系统
2. 进入 Dashboard（首页）
3. 在"今日日程"卡片中查看今天的学习安排
4. 点击"查看全部"进入完整的学习日程页面

#### 方式2：直接访问学习日程页面
1. 登录系统
2. 在侧边栏导航中找到"学习日程"（或直接访问 `/schedule`）
3. 查看和管理你的学习日程

### 4. 创建第一个日程

1. 点击"创建日程"按钮
2. 填写日程信息：
   - **标题**：例如"背英语单词"
   - **活动类型**：选择"背单词/记忆"
   - **科目**：选择"英语"
   - **日期**：选择今天或明天
   - **时间**：例如 09:00 - 10:00
   - **地点**（可选）：例如"图书馆"
   - **描述**（可选）：例如"背诵考研核心词汇500个"
3. 点击"创建"

### 5. 测试重复日程

创建一个每周重复的日程：

1. 点击"创建日程"
2. 填写基本信息
3. **重复设置**：选择"每周"
4. **重复日期**：勾选"周一"、"周三"、"周五"
5. **重复截止**：选择未来2周的日期
6. 点击"创建"

系统会自动创建所有重复日程实例！

### 6. 完成日程

1. 在今日日程列表中找到一个日程
2. 点击"完成"按钮（绿色对勾）
3. 日程状态会更新为"已完成"

### 7. 查看统计

1. 切换到"本周日程"或"本月日程"视图
2. 查看按日期分组的日程列表
3. 查看完成情况和学习时长

## 与其他功能的区别

### 学习计划 vs 学习日程

| 功能 | 学习计划 (Study Plan) | 学习日程 (Study Schedule) |
|------|---------------------|------------------------|
| 用途 | 目标管理 | 时间安排 |
| 示例 | 本月要做100道题 | 今天9:00-10:00背单词 |
| 时间粒度 | 天/周/月 | 具体时间段 |
| 关注点 | 完成数量/进度 | 时间分配/执行 |

两者互补使用，实现完整的学习管理！

## 常见问题

### Q: 如何修改已创建的日程？
A: 在日程列表中点击"编辑"按钮，修改后保存即可。

### Q: 时间冲突怎么办？
A: 系统会自动检测时间冲突，如果同一时间段已有日程，会提示错误。

### Q: 重复日程如何管理？
A: 重复日程会自动创建多个独立的日程实例，每个实例可以单独管理。

### Q: 可以删除重复日程吗？
A: 可以，但需要逐个删除每个日程实例。

### Q: Dashboard 上的今日日程卡片显示几个？
A: 最多显示5个，如果有更多会显示"还有X个日程"。
