# 笔记编辑器增强 - 题目链接简化

## 概述

根据用户反馈，将笔记编辑器的题目链接功能从基于ID的方式简化为基于标题的方式，提升用户体验和视觉美观度。

## 用户需求

1. **移除题目ID显示** - 用户做完题目后不记得ID，显示ID没有意义
2. **只显示题目标题** - 搜索结果和链接只显示题目的标题/内容
3. **改进视觉设计** - 搜索对话框更美观，链接样式更现代
4. **简化功能** - 不需要在后端绑定题目ID，只存储链接文本

## 实现的改进

### 1. 链接格式变更

**旧格式（基于ID）:**
```
[[Q:123]]
```
渲染为: 📝 题目 #123

**新格式（基于标题）:**
```
[[题:下列关于Python的描述正确的是...]]
```
渲染为: 📝 下列关于Python的描述正确的是...

### 2. 前端改进

#### 题目标题提取
新增 `getQuestionTitle()` 方法，智能提取题目标题：
- 移除 Markdown 格式字符
- 提取第一句话（以句号、问号、感叹号结尾）
- 如果太长，截取前30个字符
- 确保标题可读性

#### 链接插入优化
更新 `insertQuestionLink()` 方法：
- 使用 `[[题:标题]]` 格式替代 `[[Q:ID]]`
- 自动提取题目标题
- 插入更友好的链接文本

#### 渲染增强
更新 `renderedContent` 计算属性：
- 支持新格式 `[[题:标题]]` 的渲染
- 兼容旧格式 `[[Q:ID]]`（向后兼容）
- HTML 特殊字符转义，防止 XSS
- 移除不必要的 data 属性

#### 搜索对话框美化
- 更宽的对话框（700px）
- 更大的搜索输入框
- 移除题目ID显示
- 优化题目卡片布局：
  - 题目标题（加粗，15px）
  - 科目和章节标签
  - 内容预览（13px，灰色）

### 3. 样式优化

#### 题目链接样式
```css
.question-link-title {
  padding: 6px 14px;
  background: linear-gradient(135deg, rgba(102, 126, 234, 0.12) 0%, rgba(118, 75, 162, 0.12) 100%);
  border: 2px solid rgba(102, 126, 234, 0.3);
  border-radius: 10px;
  font-weight: 600;
  font-size: 14px;
  box-shadow: 0 2px 8px rgba(102, 126, 234, 0.1);
}

.question-link-title:hover {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  transform: translateY(-3px) scale(1.02);
  box-shadow: 0 6px 20px rgba(102, 126, 234, 0.4);
}
```

#### 搜索结果卡片
- 移除 `.question-header` 和 `.question-id`
- 新增 `.question-title` - 突出显示题目标题
- 新增 `.question-meta` - 标签容器
- 保留 `.question-preview` - 内容预览

### 4. 向后兼容

系统仍然支持旧格式的题目链接：
- `[[Q:123]]` 会渲染为 "📝 题目 #123"
- 新创建的链接使用新格式 `[[题:标题]]`
- 用户可以混合使用两种格式

## 使用示例

### 创建笔记并链接题目

1. 点击工具栏的"链接题目"按钮
2. 在搜索框输入关键词（如"Python"）
3. 从搜索结果中选择题目（只显示标题和预览）
4. 自动插入格式：`[[题:下列关于Python的描述正确的是...]]`
5. 预览时显示为漂亮的链接：📝 下列关于Python的描述正确的是...

### 链接渲染效果

**编辑模式:**
```markdown
这道题很重要：[[题:下列关于Python的描述正确的是...]]

需要复习：[[题:关于数据结构的时间复杂度...]]
```

**预览模式:**
- 这道题很重要：📝 下列关于Python的描述正确的是...
- 需要复习：📝 关于数据结构的时间复杂度...
