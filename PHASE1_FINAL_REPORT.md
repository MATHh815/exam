## 🎮 游戏化系统

### 积分规则
```
练习: 得分 + 连续奖励
考试: 得分×2 + 连续奖励
连续奖励: 天数×5
成就: 成就定义的积分
每日任务: 任务定义的积分
```

### 等级系统
```
level = floor(sqrt(total_points / 100))

等级 1: 0-399 积分
等级 2: 400-899 积分
等级 3: 900-1599 积分
等级 4: 1600-2499 积分
等级 5: 2500-3599 积分
...
```

### 成就系统
```
总成就: 24个
- 学习类: 8个
- 连续类: 6个
- 里程碑类: 10个

等级分布:
- 铜牌: 9个
- 银牌: 9个
- 金牌: 6个
```

### 每日任务
```
任务数: 5个
每日总积分: 110分

1. 完成3次练习 (20分)
2. 答对10道题目 (30分)
3. 学习30分钟 (25分)
4. 创建2条笔记 (15分)
5. 复习5道错题 (20分)
```
