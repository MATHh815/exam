## 问题和修复

### 问题 1: 测试用户不存在
**描述**: 初始测试时，测试用户 'test' 不存在  
**影响**: 登录失败，所有后续测试无法进行  
**解决方案**: 创建 create_test_user.py 脚本，自动创建测试用户  
**状态**: ✅ 已解决

### 问题 2: 密码哈希算法不匹配
**描述**: 使用 werkzeug 的 generate_password_hash 创建的密码无法验证  
**影响**: 登录失败，提示 "Invalid salt"  
**解决方案**: 改用 User 模型的 set_password() 方法，使用 bcrypt 哈希  
**状态**: ✅ 已解决
