## 🐛 常见问题

### Q1: 图表不显示？
**解决方案**:
1. 检查控制台是否有错误
2. 确认 ECharts 和 GSAP 已安装
3. 检查数据格式是否正确
4. 刷新页面重试

### Q2: 动画卡顿？
**解决方案**:
1. 关闭其他占用资源的程序
2. 检查浏览器性能
3. 减少同时运行的动画
4. 使用 Chrome 浏览器

### Q3: 数据不更新？
**解决方案**:
1. 检查 API 是否正常
2. 查看网络请求
3. 清除浏览器缓存
4. 重新登录

### Q4: 移动端显示异常？
**解决方案**:
1. 检查 viewport 设置
2. 清除浏览器缓存
3. 使用真实移动设备测试
4. 检查 CSS 媒体查询
