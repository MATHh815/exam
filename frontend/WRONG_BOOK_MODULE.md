# 错题本模块实现总结

## 概述

错题本模块已完成实现，包括 API 封装和前端页面。该模块允许用户查看、管理和练习错题。

## 实现的功能

### 1. API 封装 (`src/api/practice.js`)

实现了以下 API 函数：

- **getWrongBook(params)**: 获取错题本列表
  - 支持按掌握状态、考试类型、科目筛选
  
- **removeFromWrongBook(wrongQuestionId)**: 从错题本移除题目
  
- **startPractice(data)**: 开始练习
  - 支持从错题本练习（设置 `from_wrong_book: true`）
  
- **submitPracticeAnswer(data)**: 提交练习答案
  
- **getPracticeHistory(params)**: 获取练习历史
  
- **getPracticeSummary(params)**: 获取练习统计

### 2. 错题本页面 (`src/views/WrongBook.vue`)

#### 主要功能：

1. **错题列表展示**
   - 显示所有错题及其详细信息
   - 显示错误次数、掌握状态、最近错误时间
   - 使用 QuestionCard 组件展示题目内容

2. **筛选功能**
   - 按掌握状态筛选（未掌握/已掌握）
   - 按考试类型筛选（公务员/研究生/事业编）
   - 按科目筛选（行测/申论/数学/英语等）

3. **统计信息**
   - 错题总数
   - 未掌握题目数量
   - 已掌握题目数量

4. **错题重练功能**
   - 单题练习：在对话框中练习单个题目
   - 批量练习：跳转到练习页面，从错题本中抽题练习
   - 提交答案后自动更新掌握状态

5. **错题管理**
   - 查看/收起题目解析
   - 从错题本移除题目（带确认提示）

#### 用户体验优化：

- 加载状态提示
- 空状态提示
- 相对时间显示（刚刚、X分钟前、X小时前等）
- 响应式布局
- 操作确认提示

### 3. 路由配置

在 `src/router/index.js` 中添加了错题本路由：

```javascript
{
  path: '/wrong-book',
  name: 'wrongBook',
  component: () => import('../views/WrongBook.vue'),
  meta: { requiresAuth: true }
}
```

### 4. 导航入口

在 Dashboard 页面添加了"错题本"按钮，方便用户快速访问。

## 技术实现

### 组件复用

- 复用了 `QuestionCard` 组件展示题目
- 复用了 `AnswerOptions` 和 `QuestionExplanation` 组件

### 状态管理

- 使用 Vue 3 Composition API
- 响应式数据管理
- 计算属性优化性能

### UI 组件

使用 Element Plus 组件库：
- `el-card`: 卡片容器
- `el-form`: 筛选表单
- `el-select`: 下拉选择
- `el-statistic`: 统计数字展示
- `el-dialog`: 单题练习对话框
- `el-message-box`: 确认对话框
- `el-empty`: 空状态展示

## 与后端 API 对接

所有 API 调用都通过 `src/utils/request.js` 统一处理：
- 自动添加认证令牌
- 统一错误处理
- 响应数据格式化

## 使用方式

### 访问错题本

1. 从 Dashboard 点击"错题本"按钮
2. 直接访问 `/wrong-book` 路由

### 筛选错题

使用页面顶部的筛选表单，选择筛选条件后自动刷新列表。

### 练习错题

**单题练习**：
1. 点击题目下方的"单题练习"按钮
2. 在弹出的对话框中答题
3. 提交答案查看结果
4. 答对后自动更新掌握状态

**批量练习**：
1. 点击页面右上角的"错题重练"按钮
2. 跳转到练习页面（需要在任务 22 中实现）
3. 系统会从错题本中抽取题目

### 管理错题

- **查看解析**：点击"查看解析"按钮展开题目解析
- **移除错题**：点击"移除"按钮，确认后从错题本移除

## 数据流

```
用户操作 → Vue 组件 → API 函数 → HTTP 请求 → 后端 API
                                              ↓
用户界面 ← Vue 组件 ← 响应数据 ← HTTP 响应 ← 后端处理
```
