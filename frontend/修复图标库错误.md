# 修复 lucide-vue-next 错误

## 错误信息
```
Failed to resolve import "lucide-vue-next" from "src/components/PointsDisplay.vue"
```

---

## 快速修复

### 方法 1: 运行安装脚本（推荐）

在 `exam/frontend` 目录下运行：
```bash
install_lucide.bat
```

然后：
1. 重启 Vite 服务器（Ctrl+C 然后重新运行 `npm run dev`）
2. 刷新浏览器（F5）

---

### 方法 2: 手动安装

#### 步骤 1: 停止 Vite 服务器
在运行 `npm run dev` 的窗口按 `Ctrl+C`

#### 步骤 2: 安装依赖
```bash
cd exam\frontend
npm install lucide-vue-next
```

#### 步骤 3: 重启 Vite 服务器
```bash
npm run dev
```

#### 步骤 4: 刷新浏览器
按 `F5` 或 `Ctrl+R`

---

## 如果还是不行

### 完全重新安装依赖

```bash
cd exam\frontend

# 删除 node_modules 和 lock 文件
rmdir /s /q node_modules
del package-lock.json

# 重新安装所有依赖
npm install

# 启动开发服务器
npm run dev
```

---

## 验证安装

安装成功后，检查文件是否存在：
```bash
dir node_modules\lucide-vue-next
```

应该看到 lucide-vue-next 文件夹。

---

## 为什么会出现这个错误？

1. **依赖未安装**: 虽然 package.json 中有 `lucide-vue-next`，但 node_modules 中没有
2. **安装被中断**: 之前的 `npm install` 可能被中断或失败
3. **缓存问题**: npm 缓存可能损坏
