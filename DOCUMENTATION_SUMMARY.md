# 文档总结

本文档总结了考公考研考编系统的所有文档及其用途。

## 📚 文档列表

### 1. README.md
**用途**: 项目主文档，提供项目概览和快速开始指南

**内容**:
- 项目介绍和功能特性
- 技术栈说明
- 完整的项目结构
- 快速开始步骤
- 开发指南概览
- 测试说明
- 部署概述
- 常见问题解答

**适合人群**: 所有用户（新手、开发者、运维人员）

---

### 2. GETTING_STARTED.md
**用途**: 详细的快速开始指南

**内容**:
- 前置要求检查
- 后端设置详细步骤
- 前端设置详细步骤
- 环境变量配置
- 数据库初始化
- 常见启动问题解决方案
- 开发工作流介绍

**适合人群**: 新手开发者、首次设置项目的用户

---

### 3. API_DOCUMENTATION.md
**用途**: 完整的 API 接口文档

**内容**:
- 所有 API 端点说明
- 请求/响应格式
- 认证方式
- 错误代码说明
- 示例代码（JavaScript 和 Python）
- 速率限制说明

**API 分类**:
- 认证接口 (注册、登录、令牌管理)
- 题库接口 (CRUD、批量导入、随机抽题)
- 练习接口 (开始练习、提交答案、错题本)
- 考试接口 (试卷管理、考试会话、成绩查询)
- 统计接口 (学习概览、知识点分析、趋势图)
- 数据导入导出接口

**适合人群**: 前端开发者、API 集成开发者、测试人员

---

### 4. DEPLOYMENT.md
**用途**: 生产环境部署指南

**内容**:
- 部署架构说明
- 服务器要求和软件要求
- 数据库迁移（SQLite → PostgreSQL）
- 后端部署（Gunicorn + Systemd）
- 前端部署（静态文件）
- Nginx 配置
- Docker 部署方案
- SSL/HTTPS 配置
- 监控和日志管理
- 备份策略
- 性能优化建议
- 故障排除指南

**适合人群**: 运维人员、DevOps 工程师、系统管理员

---

### 5. DEVELOPMENT_GUIDE.md
**用途**: 开发规范和最佳实践

**内容**:
- 开发环境设置
- 项目结构详解
- 代码规范（Python 和 JavaScript）
- Git 提交规范
- 开发工作流
- 测试指南（单元测试、属性测试、组件测试）
- API 开发流程
- 数据库开发最佳实践
- 前端开发指南（Pinia、路由、API 调用）
- 调试技巧
- 常见问题解决

**适合人群**: 开发者、代码贡献者、技术负责人

---

### 6. ERROR_HANDLING_GUIDE.md (backend/)
**用途**: 错误处理系统说明

**内容**:
- 错误处理架构
- 自定义异常类
- 统一响应格式
- 验证系统
- 中间件使用
- 日志记录

**适合人群**: 后端开发者

---

### 7. DATA_IMPORT_EXPORT.md (backend/)
**用途**: 数据导入导出功能说明

**内容**:
- 数据导出功能
- 数据导入功能
- 支持的格式
- API 使用示例

**适合人群**: 数据管理员、系统管理员

---

### 8. QUICK_REFERENCE.md (backend/)
**用途**: 后端快速参考手册

**内容**:
- 常用导入语句
- 路由模板
- 异常处理模式
- Schema 示例
- 日志记录
- 测试模式

**适合人群**: 后端开发者（快速查阅）

---

## 📖 文档使用建议

### 对于新手开发者

1. 先阅读 **README.md** 了解项目概况
2. 按照 **GETTING_STARTED.md** 设置开发环境
3. 阅读 **DEVELOPMENT_GUIDE.md** 了解开发规范
4. 参考 **API_DOCUMENTATION.md** 进行开发

### 对于前端开发者

1. **README.md** - 了解项目
2. **GETTING_STARTED.md** - 设置前端环境
3. **API_DOCUMENTATION.md** - API 接口参考
4. **DEVELOPMENT_GUIDE.md** (前端部分) - 前端开发规范

### 对于后端开发者

1. **README.md** - 了解项目
2. **GETTING_STARTED.md** - 设置后端环境
3. **DEVELOPMENT_GUIDE.md** - 开发规范和最佳实践
4. **ERROR_HANDLING_GUIDE.md** - 错误处理系统
5. **QUICK_REFERENCE.md** - 快速参考

### 对于运维人员

1. **README.md** - 了解项目架构
2. **DEPLOYMENT.md** - 部署指南
3. **API_DOCUMENTATION.md** (错误代码部分) - 故障排查

### 对于项目管理者

1. **README.md** - 项目概览
2. **DEVELOPMENT_GUIDE.md** (工作流部分) - 开发流程
3. **DEPLOYMENT.md** (架构部分) - 系统架构
