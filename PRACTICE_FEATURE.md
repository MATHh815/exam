# 练习功能实现说明

## 功能概述

练习功能允许用户根据筛选条件（考试类型、题目类型、题目数量）进行随机练习，并实时查看答题结果和统计信息。

## 实现的功能

### 1. 开始练习
- ✅ 从Dashboard点击"开始练习"
- ✅ 选择筛选条件（考试类型、题目类型、题目数量）
- ✅ 后端随机抽取题目
- ✅ 跳转到练习页面

### 2. 练习页面 (`/practice`)
- ✅ 显示当前题目
- ✅ 用户选择答案
- ✅ 提交答案并判题
- ✅ 显示正确答案和解析
- ✅ 记录答题结果（正确/错误）
- ✅ 自动添加错题到错题本

### 3. 答题统计
- ✅ 实时显示答题进度
- ✅ 显示正确题数和错误题数
- ✅ 计算正确率
- ✅ 题号导航（绿色=正确，红色=错误，灰色=未答）

### 4. 练习完成
- ✅ 显示练习总结
- ✅ 统计总体表现
- ✅ 返回首页

## 使用流程

1. **进入练习**
   - 用户在Dashboard点击"开始练习"
   - 弹出对话框选择筛选条件
   - 点击"开始练习"按钮

2. **答题过程**
   - 阅读题目
   - 选择答案
   - 点击"提交答案"
   - 查看结果和解析
   - 点击"下一题"继续

3. **题号导航**
   - 右侧显示所有题号
   - 点击题号可以跳转到对应题目
   - 颜色标识：
     - 绿色：已答且正确
     - 红色：已答但错误
     - 灰色：未答

4. **完成练习**
   - 答完所有题目后点击"完成练习"
   - 查看练习总结
   - 返回首页

## API接口

### 开始练习
```
POST /api/practice/start
Request:
{
  "count": 10,
  "exam_type": "civil_service",
  "question_type": "single_choice"
}

Response:
{
  "success": true,
  "data": {
    "questions": [...],
    "count": 10
  }
}
```

### 提交答案
```
POST /api/practice/submit
Request:
{
  "question_id": 1,
  "user_answer": "A",
  "time_spent": 30
}

Response:
{
  "success": true,
  "data": {
    "is_correct": true,
    "correct_answer": "A",
    "explanation": "解析内容",
    "question": {...}
  }
}
```

## 数据流

```
Dashboard
  ↓ (选择条件)
startPracticeApi()
  ↓ (获取题目)
router.push('/practice')
  ↓ (传递题目数据)
Practice.vue
  ↓ (用户答题)
submitPracticeAnswer()
  ↓ (判题)
显示结果 + 更新统计
  ↓ (下一题/完成)
练习总结 → Dashboard
```

## 特性

### 1. 实时反馈
- 提交答案后立即显示正确/错误
- 显示正确答案和详细解析

### 2. 进度跟踪
- 实时更新答题进度

- 支持顺序答题

- 支持桌面端和移动端
- 自适应布局
