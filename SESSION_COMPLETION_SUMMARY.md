## 🔄 与现有功能的关系

### 学习计划 vs 学习日程

| 功能 | 学习计划 | 学习日程 |
|------|---------|---------|
| 用途 | 目标管理 | 时间安排 |
| 示例 | 本月要做100道题 | 今天9:00-10:00背单词 |
| 时间粒度 | 天/周/月 | 具体时间段 |
| 关注点 | 完成数量/进度 | 时间分配/执行 |

两者互补使用，形成完整的学习管理闭环！

## 📝 使用指南

### 快速开始

1. **启动服务**
   ```bash
   # 后端
   cd exam/backend
   python run.py
   
   # 前端
   cd exam/frontend
   npm run dev
   ```

2. **访问功能**
   - Dashboard查看今日日程
   - 或直接访问 `/schedule` 路由

3. **创建日程**
   - 点击"创建日程"按钮
   - 填写日程信息
   - 保存

4. **管理日程**
   - 查看：切换视图模式
   - 完成：点击完成按钮
   - 编辑：点击编辑按钮
   - 删除：点击删除按钮

详细使用指南请查看 `STUDY_SCHEDULE_QUICK_START.md`
