# 学习日程管理功能

## 功能概述

**学习日程（Study Schedule）** 是一个全新的功能，用于管理每天的具体学习安排。

### 与学习计划的区别

| 功能 | 学习计划 (Study Plan) | 学习日程 (Study Schedule) |
|------|---------------------|------------------------|
| 用途 | 目标管理 | 时间安排 |
| 示例 | 本月要做100道题 | 今天9:00-10:00背单词 |
| 时间粒度 | 天/周/月 | 具体时间段 |
| 关注点 | 完成数量/进度 | 时间分配/执行 |

## 核心功能

### 1. 日程创建

可以创建具体时间段的学习安排：

```javascript
{
  "title": "背英语单词",
  "activity_type": "memorize",  // 背单词/记忆
  "subject": "english",          // 英语
  "schedule_date": "2025-12-30",
  "start_time": "09:00",
  "end_time": "10:00",
  "location": "图书馆",
  "description": "背诵考研英语核心词汇500个"
}
```

### 2. 活动类型

支持8种学习活动：

| 类型 | 说明 | 适用场景 |
|------|------|---------|
| `memorize` | 背单词/记忆 | 背单词、记公式、背知识点 |
| `lecture` | 听课/看视频 | 网课、视频教程 |
| `practice` | 做题/练习 | 刷题、练习 |
| `review` | 复习笔记 | 复习整理的笔记 |
| `mock_exam` | 模拟考试 | 全真模拟考试 |
| `reading` | 阅读教材 | 看教材、参考书 |
| `writing` | 写作练习 | 英语作文、论述题 |
| `rest` | 休息 | 休息时间 |

### 3. 科目支持

#### 公共课
- 政治 (politics)
- 英语 (english)
- 数学 (math)

#### 专业课 - 计算机
- 数据结构 (data_structure)
- 计算机组成原理 (computer_organization)
- 操作系统 (operating_system)
- 计算机网络 (computer_network)

#### 专业课 - 其他
- 经济学 (economics)
- 管理学 (management)
- 法学 (law)
- 专业课一 (major_course_1)
- 专业课二 (major_course_2)

### 4. 重复设置

支持创建重复日程：

**单次**：
```javascript
{
  "repeat_type": "once"
}
```

**每天重复**：
```javascript
{
  "repeat_type": "daily",
  "repeat_until": "2026-01-30"  // 重复到这个日期
}
```

**每周重复**（周一到周五）：
```javascript
{
  "repeat_type": "weekly",
  "repeat_days": "1,2,3,4,5",  // 1=周一, 2=周二, ..., 7=周日
  "repeat_until": "2026-01-30"
}
```

### 5. 时间冲突检测

系统会自动检测时间冲突，防止同一时间段安排多个日程。

### 6. 提醒功能

- 可设置提前提醒时间（默认15分钟）
- 可开启/关闭提醒

### 7. 日程状态

- `pending` - 待完成
- `completed` - 已完成
- `cancelled` - 已取消

## 使用场景

### 场景1：考研日常安排

```
今天的学习安排：
08:00-09:00  背英语单词（图书馆）
09:00-11:00  听高数网课（宿舍）
11:00-12:00  休息
14:00-16:00  做政治选择题（自习室）
16:00-17:00  复习数据结构笔记
19:00-21:00  模拟考试（英语真题）
```

### 场景2：周计划

创建周一到周五的固定安排：
- 每天早上 8:00-9:00 背单词
- 每天下午 14:00-17:00 专业课学习
- 每天晚上 19:00-21:00 做题

## API 接口

### 创建日程
```
POST /api/study-schedules
```

### 获取今天的日程
```
GET /api/study-schedules/today
```

### 获取日期范围内的日程
```
GET /api/study-schedules?start_date=2025-12-30&end_date=2026-01-05
```

### 完成日程
```
PUT /api/study-schedules/:id/complete
```

### 更新日程
```
PUT /api/study-schedules/:id
```

### 删除日程
```
DELETE /api/study-schedules/:id
```

### 获取统计数据
```
GET /api/study-schedules/statistics?start_date=2025-12-01&end_date=2025-12-31
```

## 与现有功能的整合

### 1. 与学习计划整合
- 完成日程后自动更新学习计划进度
- 例如：完成"做题"日程后，自动增加"每日练习题数"目标的进度

### 2. 与番茄钟整合
- 在日程时间段内使用番茄钟
- 番茄钟完成后自动标记日程为完成

### 3. 与统计系统整合
- 日程完成情况纳入学习统计
- 各科目学习时长统计

## 数据库设计

### study_schedules 表

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| user_id | INTEGER | 用户ID |
| title | VARCHAR(200) | 标题 |
| activity_type | VARCHAR(50) | 活动类型 |
| subject | VARCHAR(50) | 科目 |
| schedule_date | DATE | 日期 |
| start_time | TIME | 开始时间 |
| end_time | TIME | 结束时间 |
| repeat_type | VARCHAR(20) | 重复类型 |
| repeat_days | VARCHAR(50) | 重复的星期几 |
| repeat_until | DATE | 重复截止日期 |
| status | VARCHAR(20) | 状态 |
| is_completed | BOOLEAN | 是否完成 |
| completed_at | DATETIME | 完成时间 |
| description | TEXT | 描述 |
| location | VARCHAR(200) | 地点 |
| reminder_minutes | INTEGER | 提醒时间 |
| is_reminder_enabled | BOOLEAN | 是否启用提醒 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |
