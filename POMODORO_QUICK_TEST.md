## 🧪 功能测试清单

### 基础功能测试

#### ✅ 1. 访问番茄钟页面
- 访问: `http://localhost:5173/pomodoro`
- 检查: 页面正常加载，显示计时器界面

#### ✅ 2. 开始番茄钟
1. 选择专注时长（默认25分钟）

4. 检查:

## 🐛 常见问题

### 问题1: 数据库表不存在
**症状**: API 返回 "no such table: pomodoro_sessions"

**解决**:
```bash
cd exam/backend
python migrate_pomodoro.py
```

### 问题2: 路由404错误
**症状**: 访问 /pomodoro 显示404

**解决**: 检查 `frontend/src/router/index.js` 是否已添加路由配置

### 问题3: API 调用失败
**症状**: 前端显示"加载数据失败"

**解决**:
1. 检查后端服务是否运行
2. 检查 JWT token 是否有效
3. 查看浏览器控制台错误信息

### 问题4: 积分未增加
**症状**: 完成番茄钟但积分没有变化

**解决**:
1. 检查 `points_service.py` 是否正常
2. 查看后端日志
3. 确认用户已登录
