# 学习计划表单修复说明

## 问题描述

创建学习计划时，表单显示多个"创建计划失败"和"缺少必填字段"错误。

## 根本原因

**前后端字段不匹配**：
- 前端表单使用：`target_date`（单个目标日期）
- 后端 API 期望：`start_date` 和 `end_date`（开始和结束日期）

## 修复内容

### 1. 更新表单字段 (`StudyPlanForm.vue`)

**修改前**：
```vue
<el-form-item label="目标日期" prop="target_date">
  <el-date-picker v-model="formData.target_date" />
</el-form-item>
```

**修改后**：
```vue
<el-form-item label="开始日期" prop="start_date">
  <el-date-picker v-model="formData.start_date" />
</el-form-item>

<el-form-item label="结束日期" prop="end_date">
  <el-date-picker v-model="formData.end_date" />
</el-form-item>
```

### 2. 更新表单数据结构

**修改前**：
```javascript
const formData = reactive({
  name: '',
  exam_type: '',
  target_date: '',  // ❌ 错误
  description: '',
  goals: [...]
})
```

**修改后**：
```javascript
const formData = reactive({
  name: '',
  exam_type: '',
  start_date: '',   // ✓ 正确
  end_date: '',     // ✓ 正确
  description: '',
  goals: [...]
})
```

### 3. 更新验证规则

**修改前**：
```javascript
const rules = {
  target_date: [
    { required: true, message: '请选择目标日期', trigger: 'change' }
  ]
}
```

**修改后**：
```javascript
const rules = {
  start_date: [
    { required: true, message: '请选择开始日期', trigger: 'change' }
  ],
  end_date: [
    { required: true, message: '请选择结束日期', trigger: 'change' }
  ]
}
```

### 4. 添加日期验证逻辑

```javascript
// 开始日期不能早于今天
const disabledStartDate = (time) => {
  return time.getTime() < Date.now() - 8.64e7
}

// 结束日期不能早于开始日期
const disabledEndDate = (time) => {
  if (formData.start_date) {
    const startTime = new Date(formData.start_date).getTime()
    return time.getTime() < startTime
  }
  return time.getTime() < Date.now() - 8.64e7
}
```

## 注意事项

### 日期格式

前端发送给后端的日期格式必须是 `YYYY-MM-DD`：
```javascript
const submitData = {
  ...formData,
  start_date: formData.start_date instanceof Date 
    ? formData.start_date.toISOString().split('T')[0]
    : formData.start_date,
  end_date: formData.end_date instanceof Date 
    ? formData.end_date.toISOString().split('T')[0]
    : formData.end_date
}
```

### 日期验证

- 开始日期不能早于今天
- 结束日期不能早于开始日期
- 两个日期都是必填的

### 考试类型

后端期望的考试类型值：
- `"公务员"` - 公务员考试
- `"考研"` - 考研
- `"事业编"` - 事业编
