## 游戏化系统

### 积分规则

```
练习: 得分 + 连续奖励
考试: 得分×2 + 连续奖励
连续奖励: 天数×5
成就: 成就定义的积分
每日任务: 任务定义的积分
```

### 等级系统

```
level = floor(sqrt(total_points / 100))

等级 1: 0-399 积分
等级 2: 400-899 积分
等级 3: 900-1599 积分
等级 4: 1600-2499 积分
等级 5: 2500-3599 积分
...
```
