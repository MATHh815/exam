## 📈 数据流程

### 完成番茄钟流程
```
用户点击"开始" 
  → 前端开始倒计时
  → 倒计时结束
  → 调用 complete_session API
  → 后端创建会话记录
  → 更新用户统计
  → 计算积分奖励
  → 检测成就解锁
  → 返回结果
  → 前端显示通知
  → 刷新统计数据
```

### 数据库操作
```
PomodoroSession (会话记录)
  ↓
PomodoroStats (统计更新)
  ↓
UserPoints (积分增加)
  ↓
Achievements (成就检测)
```
