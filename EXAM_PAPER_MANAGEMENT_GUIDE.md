# 试卷管理功能使用指南

## 问题解决

### 添加题目失败：题目不存在或已删除

**问题原因**: 数据库中没有题目数据

**解决方案**: 

1. **生成测试题目**（推荐用于测试）
   ```bash
   cd exam/backend
   python generate_test_questions.py
   ```
   这将生成10道涵盖不同类型的测试题目。

2. **导入真实题目**（用于生产环境）
   ```bash
   cd exam/backend
   python import_questions.py
   ```
   需要准备题目数据文件（JSON或Excel格式）。

3. **检查现有题目**
   ```bash
   cd exam/backend
   python check_questions.py
   ```
   查看数据库中有哪些题目可用。

## 使用流程

### 1. 创建试卷

1. 登录管理员账号
2. 进入"试卷管理"页面
3. 点击"创建试卷"按钮
4. 填写试卷信息：
   - 试卷名称：如"2024年公务员模拟考试（一）"
   - 考试类型：公务员/研究生/事业编
   - 考试时长：分钟数
   - 总分：默认100分
   - 及格分：默认60分
   - 试卷描述：可选
5. 点击"确定"创建

### 2. 添加题目到试卷

1. 在试卷列表中找到刚创建的试卷
2. 点击"管理题目"按钮
3. 在"添加题目"表单中输入：
   - **题目ID**：题目的数据库ID（1-10为测试题目）
   - **顺序**：题目在试卷中的顺序号
   - **分值**：该题目的分数
4. 点击"添加"按钮

**可用的测试题目ID**:
- ID 1-4: 公务员考试题目（行测、申论）
- ID 5-6: 研究生考试题目（英语、数学）
- ID 7-8: 事业编考试题目（公共基础知识）
- ID 9-10: 公务员考试题目（资料分析、常识判断）

### 3. 发布试卷

1. 确认试卷题目添加完成
2. 点击试卷列表中的"发布"按钮
3. 确认发布

**注意**: 试卷发布后不能再添加或修改题目！

### 4. 学生参加考试

1. 学生登录系统
2. 进入"考试"页面
3. 选择已发布的试卷
4. 点击"开始考试"
5. 答题并提交

## 常见问题

### Q1: 为什么添加题目时提示"题目不存在或已删除"？

**A**: 数据库中没有该ID的题目，或题目已被标记为删除。请先运行 `check_questions.py` 查看可用题目。

### Q2: 如何查看有哪些题目可用？

**A**: 运行以下命令：
```bash
cd exam/backend
python check_questions.py
```

### Q3: 试卷发布后能否修改？

**A**: 不能。已发布的试卷不能修改。如需修改，需要创建新版本。

### Q4: 如何批量添加题目？

**A**: 目前需要逐个添加。未来版本将支持批量导入。

### Q5: 题目顺序可以调整吗？

**A**: 可以。在添加题目时设置不同的顺序号即可。

## 数据库表结构

### questions 表（题目表）
- `id`: 题目ID
- `exam_type`: 考试类型（civil_service/postgraduate/public_institution）
- `question_type`: 题目类型（single_choice/multiple_choice/true_false/fill_blank/essay）
- `subject`: 科目
- `chapter`: 章节
- `difficulty`: 难度（1-5）
- `content`: 题目内容
- `options`: 选项（JSON）
- `correct_answer`: 正确答案
- `explanation`: 解析
- `is_deleted`: 是否删除

### exam_papers 表（试卷表）
- `id`: 试卷ID
- `name`: 试卷名称
- `exam_type`: 考试类型
- `duration`: 考试时长（分钟）
- `total_score`: 总分
- `pass_score`: 及格分
- `is_published`: 是否发布
- `version`: 版本号

### exam_paper_questions 表（试卷题目关联表）
- `id`: 关联ID
- `paper_id`: 试卷ID
- `question_id`: 题目ID
- `order`: 题目顺序
- `score`: 题目分值

## 快速开始示例

### 创建一份公务员模拟试卷

1. **创建试卷**
   - 名称：2024年公务员模拟考试（一）
   - 类型：公务员考试
   - 时长：120分钟
   - 总分：100分
   - 及格分：60分

2. **添加题目**
   ```
   题目ID: 1, 顺序: 1, 分值: 10
   题目ID: 2, 顺序: 2, 分值: 10
   题目ID: 3, 顺序: 3, 分值: 15
   题目ID: 9, 顺序: 4, 分值: 15
   题目ID: 10, 顺序: 5, 分值: 10
   ```

3. **发布试卷**
   - 点击"发布"按钮
   - 确认发布

4. **学生参加考试**
   - 学生登录后可在考试列表中看到该试卷
   - 点击"开始考试"即可参加
