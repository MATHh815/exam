# Lucide 图标导入错误修复

## 问题描述

点击"个人中心"时出现错误：
```
SyntaxError: The requested module '/node_modules/.vite/deps/lucide-vue-next.js' 
does not provide an export named 'Coin'
```

## 根本原因

`lucide-vue-next` 库中没有 `Coin` 图标（单数），正确的图标名称是 `Coins`（复数）。

## 修复内容

### 修复的文件

1. **PointsDisplay.vue** - 积分展示组件
2. **DailyTaskList.vue** - 每日任务列表组件  
3. **AchievementCard.vue** - 成就卡片组件

### 修改详情

#### 1. PointsDisplay.vue

**修改前**:
```javascript
import { Trophy, Coin, Calendar, TrendingUp } from 'lucide-vue-next'
```

**修改后**:
```javascript
import { Trophy, Coins, Calendar, TrendingUp } from 'lucide-vue-next'
```

**模板修改**:
```vue
<!-- 修改前 -->
<Coin class="stat-icon" />

<!-- 修改后 -->
<Coins class="stat-icon" />
```

#### 2. DailyTaskList.vue

**修改前**:
```javascript
import { 
  CheckSquare, Coin, Calendar, CheckCircle, TrendingUp, 
  Flame, Award, Target, BookOpen, Clock, RotateCcw 
} from 'lucide-vue-next'
```

**修改后**:
```javascript
import { 
  CheckSquare, Coins, Calendar, CheckCircle, TrendingUp, 
  Flame, Award, Target, BookOpen, Clock, RotateCcw 
} from 'lucide-vue-next'
```

**模板修改** (2处):
```vue
<!-- 修改前 -->
<Coin class="coin-icon" />

<!-- 修改后 -->
<Coins class="coin-icon" />
```

#### 3. AchievementCard.vue

**修改前**:
```javascript
import { 
  Trophy, Award, Star, Target, Flame, BookOpen, 
  CheckCircle, TrendingUp, Lock, Coin, Calendar 
} from 'lucide-vue-next'
```

**修改后**:
```javascript
import { 
  Trophy, Award, Star, Target, Flame, BookOpen, 
  CheckCircle, TrendingUp, Lock, Coins, Calendar 
} from 'lucide-vue-next'
```

**模板修改**:
```vue
<!-- 修改前 -->
<Coin class="coin-icon" />

<!-- 修改后 -->
<Coins class="coin-icon" />
```

## 如何避免类似问题

### 1. 查看 Lucide 文档

访问 [Lucide Icons](https://lucide.dev/icons/) 查看所有可用图标

### 2. 使用 TypeScript

TypeScript 会在编译时检查导入的图标是否存在

### 3. 测试导入

在使用新图标前，先测试导入：
```javascript
import { IconName } from 'lucide-vue-next'
```

如果图标不存在，Vite 会立即报错

## 常见的 Lucide 图标名称规则

1. **复数形式**: 某些图标使用复数（如 `Coins` 而不是 `Coin`）
2. **驼峰命名**: 所有图标使用 PascalCase（如 `TrendingUp`）
3. **完整单词**: 通常使用完整单词而不是缩写
