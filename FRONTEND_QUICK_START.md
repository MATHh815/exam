# 前端功能快速开始指南

## 概述
本指南帮助你快速测试新开发的学习计划、笔记管理和题目收藏功能。

---

## 前置条件

### 1. 后端服务运行
```bash
cd exam/backend
python run.py
```
后端服务应该运行在 `http://localhost:5000`

### 2. 前端服务运行
```bash
cd exam/frontend
npm run dev
```
前端服务应该运行在 `http://localhost:5173`

### 3. 用户账号
确保你有一个已注册的用户账号，或者注册一个新账号。

## 功能测试指南

### 1. 学习计划管理

#### 访问路径
登录后，点击侧边栏 **学习工具 > 学习计划**，或直接访问 `/study-plans`

#### 测试步骤

**创建学习计划**:
1. 点击右上角"创建计划"按钮
2. 填写表单：
   - 计划名称：`2024年公务员备考计划`
   - 考试类型：选择"公务员考试"
   - 目标日期：选择未来的日期
   - 计划描述：`全面备考公务员考试`
3. 添加学习目标：
   - 目标类型：每日练习题数
   - 目标值：50
4. 点击"添加目标"可以添加更多目标
5. 点击"创建计划"提交

**查看计划列表**:
- 在"进行中"标签页查看活跃计划
- 在"已完成"标签页查看已完成计划
- 每个计划卡片显示：
  - 计划名称和状态
  - 考试类型和目标日期
  - 学习目标和进度条

**查看计划详情**:
1. 点击计划卡片上的"查看"图标
2. 查看完整的计划信息和目标列表

**编辑计划**:
1. 点击计划卡片上的"编辑"图标
2. 修改计划信息
3. 点击"更新计划"保存

**查看学习报告**:
1. 点击计划卡片底部的"查看报告"按钮
2. 查看学习统计数据：
   - 总练习题数
   - 总考试次数
   - 总学习时长
   - 目标完成情况

**删除计划**:
1. 点击计划卡片上的"删除"图标
2. 确认删除操作

### 2. 笔记管理

#### 访问路径
登录后，点击侧边栏 **学习工具 > 我的笔记**，或直接访问 `/notes`

#### 测试步骤

**创建笔记**:
1. 点击右上角"新建笔记"按钮
2. 填写表单：
   - 题目ID：输入一个存在的题目ID（例如：1）
   - 标签：选择或自定义标签（例如：重点、易错）
   - 笔记内容：使用 Markdown 格式编写笔记
3. 使用工具栏快捷插入 Markdown 标记：
   - **B**: 加粗
   - *I*: 斜体
   - `</>`: 代码
   - H1/H2/H3: 标题
   - 列表和引用
4. 切换"预览"开关查看渲染效果
5. 点击"创建"提交

**Markdown 示例**:
```markdown
# 解题思路

这道题的关键是理解**核心概念**。

## 步骤

1. 分析题目条件
2. 列出已知信息
3. 推导结论

> 注意：这是一个易错点

代码示例：`x = y + z`
```

**搜索笔记**:
1. 在搜索框输入关键词
2. 使用筛选器：
   - 科目筛选
   - 排序方式（最新创建、最早创建、最近更新）
3. 点击搜索或按回车

**查看笔记**:
1. 点击笔记卡片
2. 查看完整的笔记内容（Markdown 渲染）
3. 查看标签和元信息

**编辑笔记**:
1. 在笔记详情对话框点击"编辑"
2. 或点击笔记卡片右上角的"..."菜单，选择"编辑"
3. 修改笔记内容
4. 点击"更新"保存

**删除笔记**:
1. 在笔记详情对话框点击"删除"
2. 或点击笔记卡片右上角的"..."菜单，选择"删除"
3. 确认删除操作

### 3. 题目收藏

#### 访问路径
登录后，点击侧边栏 **学习工具 > 我的收藏**，或直接访问 `/bookmarks`

#### 测试步骤

**收藏题目**:
> 注意：收藏功能需要在练习或考试页面中使用，这里只能查看和管理已收藏的题目

在练习页面收藏题目（需要集成）:
1. 在题目页面点击"收藏"按钮
2. 添加标签和备注
3. 确认收藏

**查看收藏列表**:
1. 访问收藏页面
2. 查看所有收藏的题目
3. 每个收藏卡片显示：
   - 题目信息（类型、难度、科目）
   - 题目内容
   - 标签和备注
   - 收藏时间

**筛选收藏**:
使用筛选器：
- 考试类型（公务员、考研、事业编）
- 难度（简单、中等、困难）
- 标签搜索
- 排序方式（最新收藏、最早收藏、难度升序、难度降序）

**查看题目详情**:
1. 点击收藏卡片底部的"查看题目"按钮
2. 查看完整的题目信息：
   - 题目内容
   - 选项列表（标记正确答案）
   - 正确答案
   - 解析

**编辑收藏**:
1. 点击收藏卡片底部的"编辑收藏"按钮
2. 修改标签和备注
3. 点击"更新"保存

**取消收藏**:
1. 点击收藏卡片右上角的"取消收藏"按钮
2. 确认取消操作

## 常见问题

### 1. 创建学习计划时提示"目标值必须大于0"
**原因**: 目标值必须是正整数  
**解决**: 确保目标值大于0

### 2. 笔记内容无法保存
**原因**: 笔记内容超过5000字符限制  
**解决**: 减少笔记内容长度，或拆分为多条笔记

### 3. 收藏列表为空
**原因**: 还没有收藏任何题目  
**解决**: 在练习或考试页面收藏题目（功能待集成）

### 4. Markdown 预览不显示
**原因**: 浏览器兼容性问题  
**解决**: 使用现代浏览器（Chrome、Firefox、Edge）

### 5. 搜索没有结果
**原因**: 搜索关键词不匹配  
**解决**: 尝试使用不同的关键词或清除筛选条件

## 开发者提示

### API 调用
所有 API 调用都需要 JWT 认证，确保：
1. 用户已登录
2. Token 存储在 localStorage 中
3. 请求头包含 `Authorization: Bearer <token>`

### 调试技巧
1. 打开浏览器开发者工具（F12）
2. 查看 Network 标签页的 API 请求
3. 查看 Console 标签页的日志输出
4. 使用 Vue DevTools 查看组件状态

### 数据持久化
- 学习计划、笔记、收藏数据都存储在后端数据库
- 刷新页面不会丢失数据
- 删除操作是软删除，数据不会真正删除
