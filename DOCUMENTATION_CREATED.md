   ```bash

   cat exam/ENHANCEMENTS_INDEX.md
   
   # 或在编辑器中打开
   code exam/ENHANCEMENTS_INDEX.md
   ```

2. **使用 Kiro 创建设计文档**
   
   在 Kiro 中输入：
   ```

   ```

   设计文档完成后：
   ```
   请基于设计文档创建任务列表
   ```

   任务列表创建后：
   ```
   请执行第一个任务：创建 StudyPlan 数据模型
   ```
